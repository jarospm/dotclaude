# bird

CLI tool for reading and posting to Twitter/X using browser cookies for authentication.

## Source

- GitHub: https://github.com/steipete/bird
- Install: `brew install steipete/tap/bird` or `npm install -g @steipete/bird`

## Prerequisites

- Node.js or Homebrew
- Logged into x.com in Chrome, Safari, or Firefox

## Configuration

Config file: `~/.config/bird/config.json5`

Current setup — uses Chrome "Profile 2" for auth:

```json5
{
  chromeProfile: "Profile 2"
}
```

Other config options:

- `cookieSource` — browser to use (chrome, safari, firefox)
- `timeoutMs` — request timeout
- `quoteDepth` — max quoted tweet depth (default: 1)

Can also use env vars `AUTH_TOKEN` and `CT0` for manual credentials.

## Quick Reference

```bash
# Account
bird whoami                      # Show logged-in account

# Reading
bird read <tweet-id-or-url>      # Fetch tweet content
bird thread <id>                 # Show full conversation thread
bird replies <id>                # List tweet replies
bird search "query"              # Search tweets
bird user-tweets @handle         # Get profile timeline

# Social
bird following                   # List users you follow
bird followers                   # List your followers
bird mentions                    # Tweets mentioning you

# Bookmarks & Likes
bird bookmarks                   # List bookmarked tweets
bird likes                       # View liked tweets

# News
bird news                        # AI-curated news from Explore
bird trending                    # Alias for news

# Write operations
bird tweet "text"                # Post a new tweet
bird reply <id> "text"           # Reply to a tweet
```

Common flags: `--json`, `--plain` (script-friendly output without emoji)
