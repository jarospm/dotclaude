# /last30days — Claude Code Skill

Researches the last 30 days of Reddit and X discussions
on any topic, then synthesizes findings into actionable
prompts or expert summaries.

## What It Does

- Scans Reddit and X for recent discussions on a topic
- Identifies patterns and best practices by engagement
- Generates copy-paste prompts or expert-level summaries
- Typical execution: 2-8 minutes

## Example Usage

- `/last30days prompting techniques for ChatGPT`
- `/last30days top Claude Code skills`
- `/last30days Nano Banana Pro prompting`

## Why It's Interesting

Grounds responses in recent, upvoted community content
rather than outdated docs — useful for fast-moving topics.
