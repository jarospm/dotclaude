## Description

A collection of AI agent skills — packaged instructions and automation scripts that extend coding agent capabilities. Designed for Claude Desktop and claude.ai.

### Available Skills

- **React Best Practices** — 40+ rules across 8 categories for optimizing React/Next.js apps (waterfalls, bundle size, SSR, data fetching, re-renders)
- **Web Design Guidelines** — audits UI code against 100+ rules covering accessibility, performance, and UX
- **Vercel Deploy Claimable** — deploys apps to Vercel directly from chat, auto-detects 40+ frameworks

### Key Features

- Skills activate automatically based on detected tasks
- Modular structure with SKILL.md instructions, helper scripts, and docs
- Framework auto-detection across dozens of popular tools

## Why Interested

High-quality reference implementation of the skills format. The React best practices and web design guidelines could be useful references or directly integrated. The deploy skill shows a pattern for packaging complex automation.
