# x-article-publisher-skill

URL: https://github.com/wshuyi/x-article-publisher-skill

## What it does

Claude Code skill for publishing Markdown articles to X (Twitter) Articles via browser automation. Converts a 20-30 minute manual process into 2-3 minutes.

**Features:**

- Parses Markdown and extracts titles, cover images, content images
- Converts formatting to HTML while preserving styling
- Uses Playwright for browser automation with block-index positioning
- Supports dividers, tables, and Mermaid diagrams
- Creates drafts only (never auto-publishes)

**Dependencies:**

- Python 3.9+
- Claude Code with Playwright MCP
- X Premium Plus account (required for Articles)
