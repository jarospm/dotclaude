# Superpowers

Workflow framework that enforces methodical AI agent development process.

**Repo:** https://github.com/obra/superpowers

## What It Does

- Structured workflow: design → plan → TDD → review
- Composable skills that trigger at specific development stages
- Interactive brainstorming before coding
- Test-first enforcement
- Built-in code review and git branch management

## Philosophy

- Don't write code immediately — understand first
- Present design in digestible chunks for approval
- Test-driven development as default
- Verification over assumption

## Integration

- Works with Claude Code as a plugin
- Also supports Codex and OpenCode
- MIT licensed

## Why Interesting

- More structured than ad-hoc prompting
- Enforces discipline that's easy to skip manually
- Could complement or replace current workflow patterns
