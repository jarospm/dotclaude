# Agent OS

Lightweight system for keeping AI agents aligned with codebase standards.

**Repo:** https://github.com/buildermethods/agent-os

## What It Does

- Discovers patterns from codebases → documented standards
- Deploys standards contextually during development
- Shapes specs for better planning
- Indexes standards for retrieval when needed

## Why Interesting

- Complementary to Claude Code — not a replacement
- Could improve consistency on larger projects
- Structured approach to knowledge management for AI workflows

## Integration Notes

- Uses YAML config + shell scripts
- Works alongside existing AI tools (Claude Code, Cursor, etc.)
- Four commands: discover, deploy, shape, index
