# Happy Engineering

Open-source mobile and desktop client for Claude Code.

## What It Is

Control Claude Code from your phone. Spawn and manage multiple Claude Code
sessions in parallel from iOS, Android, or web.

## Installation

```bash
npm i -g happy-coder && happy
```

Also available on iOS and Android app stores.

## Features

- Voice-controlled coding (hands-free)
- Multiple simultaneous Claude Code sessions
- End-to-end encryption
- Remote access to terminal workflows

## Use Cases

- Running Claude Code from mobile when away from desk
- Voice-driven development
- Managing multiple Claude sessions across machines
- Hands-free coding while doing other tasks
