# Skills.sh

Open ecosystem directory for AI agent skills.

## What It Is

Discovery platform and installation registry for reusable AI agent capabilities.
Browse 200+ skills, install with a single command.

## Features

- One-command install: `npx skills add <owner/repo>`
- Multi-agent support: Claude Code, Copilot, Cursor, Gemini, others
- Leaderboard by popularity
- Searchable catalog

## Use Cases

- Finding pre-built skills instead of writing from scratch
- Discovering workflows other power users have created
- Skill inspiration and patterns
- Comparing popular approaches
