# Conductor

macOS app for running multiple AI coding agents in parallel.

## What It Is

Desktop application that manages concurrent Claude Code and Codex agents
in isolated workspaces using git worktrees.

## Features

- Spawn multiple agents working on different tasks simultaneously
- Each agent runs in isolated environment
- Unified interface to monitor progress
- Review and merge changes back to main

## Pricing

Uses your existing auth (Claude API key, Pro, or Max subscription).

## Use Cases

- Parallelizing coding work across multiple features
- Running agents on different repos simultaneously
- Accelerating development throughput
